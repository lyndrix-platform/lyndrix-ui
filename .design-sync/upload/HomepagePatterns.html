Plugin-First Architecture

Develop only what makes your app unique. Drop in a plugin with setup(ctx) and the framework handles the rest.

Vault-Backed Secrets

First-class HashiCorp Vault integration with automatic bootstrap, unseal, and per-plugin secret isolation. Zero-trust by default.

Event-Driven Core

A global event bus decouples every component. Plugins communicate through declared topics — no spaghetti imports, no tight coupling.

Write Your Plugin

Create entrypoint.py with a manifest and setup(ctx). That's the entire contract.

04
You're Done

Your plugin is live. Managed from the built-in Plugin Manager. No restarts, no manual wiring.

entrypoint.py Python 
from core.api import ModuleManifest
from nicegui import ui

manifest = ModuleManifest(
    id="lyndrix.plugin.myapp",
    name="My App",
    version="1.0.0",
    type="PLUGIN",
    permissions={
        "subscribe": ["system:boot_complete"],
        "emit": [],
    },
)

async def setup(ctx):
    @ui.page('/myapp')
    async def page():
        ui.label('Hello!')
